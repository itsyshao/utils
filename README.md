# utils
JAVA工具集合类，方便项目集成应用

#JschUtils
实现远程服务器的附件上传，执行单条命令，动态执行脚本。
* 依赖包：commons-lang3-3.5.jar jsch-0.1.54.jar

#ExcelUtils
实现EXCEL与实体对象的相互转换
* 依赖包：commons-lang3-3.5.jar poi-3.15.jar servlet-api.jar

#XmlUtils
实现XML与MAP互相转换
* 依赖包：commons-lang3-3.5.jar dom4j-1.6.1.jar

#JsonUtils
实现XML与JSON互相转换
* 依赖包：json-20160212.jar

#DateUtils
日期工具类，格式转换，特定日期获取
